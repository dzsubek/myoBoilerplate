myoBoilerplate
==============

Basic Gestures (basicGestures.myo)
--------------

#### Events
* onRest
  * onRestStart
  * onRestEnd
* onFist
  * onFistStart
  * onFistEnd
* onWaveIn*
  * onWaveInStart
  * onWaveInEnd
* onWaveOut*
  * onWaveOutStart
  * onWaveOutEnd
* onFingersSpread
  * onFingersSpreadStart
  * onFingersSpreadEnd
* onDoubleTap
  * onDoubleTapStart
  * onDoubleTapEnd

*You can switch onWaveIn and onWaveOut events when myo is on left arm with change `switchLeftHand`  from `false` to `true`.
